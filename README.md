- Walkthorugh link: https://youtu.be/jsNxCIx1GVU

## Getting Started

First, run the development server:

```bash
# install deps
npm install
# or
yarn

# run locally
npm run dev
# or
yarn dev
```
Open [http://localhost:3000](http://localhost:3000)
